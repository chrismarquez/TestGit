Product Evaluation: Transport ways for Canadian foreigners in Mexico

The following products are solutions for foreigners to adapt and move easier on mexican roads.

The first product is a teleporting device whcih allows the user to enter a gate on one side of the country and get instantly to the place the user desires to go to.

The next product is a mexican roads handbook which allows the user to know which roads are nicer to take so the traveller can get in an easier way to his or her destiny

Teleport Device
Advantages Disadvantages

Roads Handbook
Advantages Disadvantages
It's cheap and easy to use It doesn't work so well while driving

It can be distributed to most of the population

  • Suggestions for both products
    • Instead of a book, publish a weekly magazine so it can have recent information.
    • Even though theoretically teletransportation is possible, it cannot be implemented in the near future.
